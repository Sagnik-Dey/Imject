### Languages used
- Python
- CSS

### How to run
- Clone the repository
- Install the required libraries
  ```bash
  pip install -r requirements
  ```
- Change directory to src
  ```bash
  cd src
  ```
- Run main.py
  ```bash
  python main.py
  ```
